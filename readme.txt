Установка:

!Перед установкой, убелитесь что у Вас устаовленна актуальная версия node.js!

1. Скачать архив по ссылке на репозиторий (https://github.com/starnizar/NewLandTest/archive/refs/heads/master.zip)
или (при наличии gitBush) ввести в команду 'git clone https://github.com/starnizar/NewLandTest.git'

2. Открыть проект в среде разработки

3. В терминале ввести команду 'npm install' для установки всех пакетов использованных при разработке

4. Затем запустить проект в режиме разработки, введя команду 'npm start'

5. Убедившись, что нет ошибок при работе, можно создать оптимизированное приложение для публикации, введя команду 'npm run build'
